# AI-EVA Launcher 安装说明

## 前置要求

1. **Node.js 18+**
   - 下载地址: https://nodejs.org/
   - 验证安装: `node --version`

2. **Rust 1.70+**
   - 下载地址: https://www.rust-lang.org/tools/install
   - 安装命令: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
   - 验证安装: `rustc --version`

3. **Git** (用于克隆 IndexTTS2)
   - 下载地址: https://git-scm.com/downloads
   - 验证安装: `git --version`

4. **系统依赖** (Windows)
   - Microsoft Visual C++ Build Tools
   - Windows SDK

## 安装步骤

### 1. 安装项目依赖

```bash
cd AI-EVA-Launcher
npm install
```

### 2. 开发模式运行

```bash
npm run tauri dev
```

### 3. 构建生产版本

```bash
npm run tauri build
```

构建完成后，可执行文件位于:
- Windows: `src-tauri/target/release/ai-eva-launcher.exe`
- macOS: `src-tauri/target/release/ai-eva-launcher.app`
- Linux: `src-tauri/target/release/ai-eva-launcher`

## 图标文件

需要在 `src-tauri/icons/` 目录下放置以下图标文件：

- `32x32.png` - 32x32 像素 PNG
- `128x128.png` - 128x128 像素 PNG
- `128x128@2x.png` - 256x256 像素 PNG (Retina)
- `icon.icns` - macOS 图标
- `icon.ico` - Windows 图标

如果没有图标文件，应用仍可正常运行，但打包时会缺少图标。

## 常见问题

### Q: Rust 编译失败？
A: 确保已安装 Rust 工具链，运行 `rustup update` 更新到最新版本。

### Q: npm install 失败？
A: 确保 Node.js 版本 >= 18，尝试清除缓存: `npm cache clean --force`

### Q: Git clone 失败？
A: 确保已安装 Git，并且网络可以访问 GitHub。

### Q: 下载速度慢？
A: 可以配置代理或使用镜像源。

## 使用说明

1. 启动应用后，输入你的 GitHub 仓库地址（例如: `https://github.com/your-username/AI-EVA`）
2. 选择或输入安装路径（默认: `~/AI-EVA`）
3. 点击"开始下载"按钮
4. 等待下载和克隆完成
5. 点击"启动 AI-EVA"按钮启动应用

## 技术说明

- **前端**: 纯 HTML/CSS/JavaScript，使用 Tauri API 与后端通信
- **后端**: Rust，处理文件下载、Git 克隆、进程启动等操作
- **下载**: 使用 reqwest 库下载 GitHub 仓库的 ZIP 文件
- **解压**: 使用 zip 库解压下载的文件
- **Git 克隆**: 调用系统 Git 命令克隆 IndexTTS2 仓库
