# 快速创建占位符图标

## 方法 1: 使用在线工具（推荐）

访问以下网站，上传任意图片，生成所需图标：

1. **ICO 文件生成器**: https://www.icoconverter.com/
   - 上传任意图片（建议 256x256 或更大）
   - 选择生成多尺寸 ICO 文件
   - 下载并保存为 `icon.ico`

2. **PNG 图标生成器**: https://www.favicon-generator.org/
   - 上传图片
   - 下载生成的 PNG 文件
   - 重命名为: `32x32.png`, `128x128.png`, `128x128@2x.png` (256x256)

## 方法 2: 使用 Python 脚本

如果已安装 Python 和 Pillow：

```bash
cd src-tauri
python create_placeholder_icons.py
```

或运行批处理文件：

```bash
create_icons.bat
```

## 方法 3: 手动创建最小图标

### Windows ICO 文件

可以使用以下 PowerShell 命令创建一个最小的 ICO 文件（需要 ImageMagick 或类似工具）：

```powershell
# 如果安装了 ImageMagick
magick convert -size 256x256 xc:"#667eea" icon.ico
```

### PNG 文件

可以使用任何图片编辑软件创建：
- 32x32.png - 32x32 像素
- 128x128.png - 128x128 像素  
- 128x128@2x.png - 256x256 像素（实际文件名）

## 临时解决方案

如果只是想快速测试，可以：

1. 从网上下载一个免费的图标
2. 使用在线转换工具转换为所需格式
3. 放置到 `icons/` 目录

## 图标要求

- **格式**: PNG (透明背景) 或 ICO
- **尺寸**: 见上方说明
- **内容**: 可以是简单的文字、图形或 Logo

## 推荐图标资源

- https://www.flaticon.com/ (免费图标)
- https://icons8.com/ (免费图标)
